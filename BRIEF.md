## 4. 技術スタック

### 確定スタック
```
フロント:    Next.js 15 (App Router) + TypeScript
スタイル:    Tailwind CSS
ホスティング: Vercel (仮ドメイン: *.vercel.app 自動発行)
フォーム:    Google Apps Script (GAS) → Google スプレッドシート
日程調整:    TimeRex iframe 埋め込み
計測:        Google Tag Manager (GTM) 経由で GA4 + Meta Pixel
```

### 選定理由
- **Next.js**: 診断の出し分け・ページ遷移・SEO に強く、求人業界での採用実績が豊富
- **Tailwind**: ワイヤー忠実な見た目を高速に実装できる
- **Vercel**: Next.js 公式相性、無料枠で十分、SSL 自動、表示速度世界最速クラス
- **GAS + スプシ**: サーバー不要、無料、運用者が直接スプシでデータ確認可能
- **TimeRex iframe**: 30分で実装完了。VibesRadar が手動送信である以上、API 一体化のメリットが薄い
- **GTM**: GA4 だけでなく Meta 広告ピクセルや今後の追加タグを Claude/開発不在でも管理画面から追加可能
